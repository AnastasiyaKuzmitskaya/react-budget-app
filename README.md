### Installation

###### 1. Download or clone repo : 
> _https://github.com/AnastasiyaKuzmitskaya/react-budget-app.git_
###### 2. Go to unzipped folder 
###### 3. Install packages with :
> _npm install_
###### 4. Run the application : 
> _npm start_
